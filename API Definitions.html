get /airQualityByCoordinatesAndRadiusAndMeasuredParam
Get AirQuality data points by coordinates, radius, and measured param (airQualityByCoordinatesAndRadiusAndMeasuredParamGet)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Query parameters

latitude (required)
Query Parameter — decimal degree coordinate
longitude (required)
Query Parameter — decimal degree coordinate
radius (required)
Query Parameter — decimal degree coordinate format: int32
measuredParam (required)
Query Parameter — parameter name of target measurement

Return type

ApiSuccessResponse

{"empty": false}
